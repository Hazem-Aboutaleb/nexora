# Project Overview

Nexora is an AI Intelligence Layer built on top of existing CCTV systems.

The customer should never replace:

- Cameras
- DVR
- NVR
- Network infrastructure

Nexora connects to existing systems and transforms recorded video into searchable and analyzable information using Artificial Intelligence.

---

# Current Goal

Build the first MVP.

The MVP includes only:

1. Connect to DVR/NVR.
2. Display available cameras.
3. Read recorded videos.
4. AI Search.
5. Jump directly to matching timestamps.

Nothing else is a priority.
