# PROJECT CONTEXT

## Overview

Nexora is an AI-powered software platform that transforms traditional surveillance systems into intelligent business tools.

The project does not replace cameras, DVRs, or existing infrastructure.

Instead, it adds an intelligence layer capable of understanding video content.

---

# The Problem

Millions of surveillance cameras record video every day.

Almost none of this data is actually used.

Businesses only watch recordings after an incident occurs.

Searching for a specific event can take hours.

Traditional surveillance systems record.

They do not understand.

---

# The Solution

Nexora connects to existing surveillance systems and analyzes recorded video using Artificial Intelligence.

Instead of manually reviewing footage, users can search using natural language.

Example:

- Find the man wearing a blue shirt.
- Show everyone entering after 8 PM.
- Find customers carrying backpacks.

Nexora returns the exact recording and timestamp.

# MVP

Version 1 includes only:

1. Connect to DVR/NVR.
2. Read recorded videos.
3. Display available cameras.
4. Search using natural language.
5. Jump directly to matching timestamps.

Everything else is postponed until the MVP succeeds.
